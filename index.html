Basal

Basal is a lightweight, cross-browser framework for web projects. The main goal is to provide a drop-in unobtrusive modern framework to start designing a site right away, with the use SASS, a powerful CSS preprocessor.

Getting started

Download Basal, then run npm install (assuming you have Node.js installed on your system) in the root directory of the project. You're ready to go.
